# VesperLauncher
 VesperLauncher 是一款自研的《我的世界》Java版第三方启动器，名称中“Vesper”意为黄昏、暮星，取自项目构思于傍晚的灵感，寓意在日暮开启像素世界的旅程，中文可译作暮启启动器。   本启动器主打轻量化、易用性与可拓展性，面向国内MC玩家与模组爱好者。程序注重低内存占用、快速启动，UI适配黄昏风格与深色模式，视觉柔和。支持微软正版账号登录，严格遵循游戏用户协议，不提供盗版相关功能。   核心规划包含多游戏版本管理、Forge/Fabric适配、模组与资源文件管理、可视化JVM参数配置、日志查看及版本备份等功能，同时预留拓展接口，便于二次开发。可作为个人学习项目或面向玩家的实用启动工具。
